# 基本操作

## 基本命令
git clone url
git checkout -b T1
git checkout T1
git checkout .
git branch 
git branch -D T1
git branch -m T1
git status
git diff
git add .
git commit [-m 'message']
git push origin T1
git fetch origin T1:T1
git merge --no-ff master-branch
git reset --mixed HEAD^

## 代码冲突解决流程
1. git checkout test
2. git pull origin test
3. git checkout 任务分支
4. git checkout -b 任务分支-1
5. git merge --no-ff test
6. 代码中解决冲突
7. git add .
8. git commit
9. git push origin 任务分支-1
10 .gitlab上合并 任务分支-1 到 test
